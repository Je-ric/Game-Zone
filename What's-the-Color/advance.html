How to Play

Game Modes

  • RGB Mode: Guess which color matches the RGB value
  • HEX Mode: Guess which color matches the HEX code
  • Time Attack: Make as many correct guesses as possible in 30 seconds
  • Streak Mode: See how many correct guesses you can make in a row

Difficulty Levels

  • Easy: 3 colors to choose from
  • Medium: 6 colors to choose from
  • Hard: 9 colors to choose from

Click on the color you think matches the displayed color code. The more difficult the level, the more points you earn for correct guesses!
